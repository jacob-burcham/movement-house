# The Studio

A marketing website for **The Studio**, a membership-based movement studio offering dance, yoga, and calisthenics classes.

## Tech Stack

- **React 18** with TypeScript
- **Vite 6** for development and builds
- **Tailwind CSS v4** for styling
- **React Router DOM** for client-side routing
- **shadcn/ui** + Radix UI for accessible components

## Quick Start

```bash
npm install
npm run dev
```

## Features

- Landing page with hero, about, classes, membership, amenities, and contact sections
- Waitlist signup form with membership tier selection
- Responsive design with mobile navigation
- Brand-consistent design system (yellow, red, purple palette)
